RESPONSIVE GRID

Based on a 12 column grid. Simply create a div with a class name corresponding to how large you would like the grid section to appear on a small screen (480px or less). Then, apply more classes corresponding to how you would like the section to appear on your arbitrary breakpoints, all the way up to extra large screens (1200px and up). Experiment with the examples below by changing the size of your browser window to get an idea of how these breakpoints work.

Driven by jQuery and GreenSock Animation Platform. Simply add the corresponding class to a div to turn it into a stylized button. Javascript will inject the appropriate styling in line with to the size of the content in the button, so you will never have to calculate any tricky proportions.

BE AWARE OF A FEW THINGS, THOUGH. Some of these styles actively change size on hover. Make sure it does not cause line breaking or exceed the width of its containing element, unless that is your intention.

The best way to make these is to make sure that whatever element you want to base the button's proportions off of is marked up as the first child element of the button. Also, wrapping the entire button in an anchor tag ensures that it can be selected if the user clicks on any part of the button.
